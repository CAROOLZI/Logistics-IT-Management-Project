# Integrated Management System: Logistics & IT Assets

This repository demonstrates an integrated solution for **Supply Chain Logistics** and **IT Asset Management**, combining database modeling, cloud architecture concepts, and data analysis.

## 🛠 Tech Stack & Skills
- **Database:** SQL (Oracle Standards & Fundação Bradesco Best Practices)
- **Programming:** Python for data processing and automation
- **Cloud:** AWS (Architecting solutions using RDS, S3, and Lambda)
- **Business Intelligence:** Power BI for KPI visualization

## 📂 Project Structure
- `/database`: SQL scripts for table creation and modeling.
- `/scripts`: Python automation for data validation.
- `/cloud`: Documentation of the AWS infrastructure proposal.

## ☁️ AWS Architecture Proposal (AWS Educate)
To ensure scalability, this system is designed to run on:
1. **Amazon RDS:** Hosting the relational database.
2. **AWS Lambda:** Running serverless scripts to validate cargo weight (e.g., Fiat Toro/S10 capacity).
3. **Amazon S3:** Storing automated logs and reports.

# Sistema de Gestão Integrada: Logística e Ativos de TI

Este repositório demonstra uma solução integrada para **Logística de Cadeia de Suprimentos** e **Gestão de Ativos de TI**, combinando modelagem de banco de dados, conceitos de arquitetura em nuvem e análise de dados.

## 🚀 Visão Geral
O projeto simula um cenário real para uma grande empresa do setor de Bens de Consumo (FMCG), onde o monitoramento da frota de entrega e da infraestrutura técnica é crucial para o sucesso operacional.

## 🛠 Tecnologias e Habilidades
- **Banco de Dados:** SQL (Padrões Oracle e Boas Práticas da Fundação Bradesco)
- **Programação:** Python para processamento de dados e automação
- **Nuvem:** AWS (Arquitetura de soluções utilizando RDS, S3 e Lambda)
- **Business Intelligence:** Power BI para visualização de KPIs

## 📂 Estrutura do Projeto
- `/database`: Scripts SQL para criação de tabelas e modelagem.
- `/scripts`: Automação em Python para validação de dados.
- `/cloud`: Documentação da proposta de infraestrutura AWS.

## ☁️ Proposta de Arquitetura AWS (AWS Educate)
Para garantir a escalabilidade, este sistema foi projetado para rodar em:
1. **Amazon RDS:** Hospedagem do banco de dados relacional.
2. **AWS Lambda:** Execução de scripts Python (Serverless) para validar a carga dos veículos (ex: capacidade da Fiat Toro/S10).
3. **Amazon S3:** Armazenamento de logs e relatórios automatizados.
